## ✅为什么选 Hexo？
- 核心优势：轻量、静态、Markdown 写作、免费部署（GitHub Pages）
- 适用人群：程序员、内容创作者、想低成本搭个人博客的**新手**

## ✅前置准备
- Node.js
    ```bash
        # 验证Node.js版本
        node -v
        # 验证npm版本（随Node自动安装）
        npm -v
    ```
- Git (版本控制 + 部署必备)
    ```bash
        # 验证Git版本
        git --version
    ```
- 开发工具 Cursor
